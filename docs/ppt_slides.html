2. Business Understanding 商業理解 Slide 2 / 8

💼 商業背景

創業投資機構 (VC) 與創業決策者需要評估新創公司的各項支出配置對最終盈餘 (Profit) 的推動力，以決定是否投資，或是優化內部財務開支比例。

🎯 分析任務與演算法

本專案建立一個「多元線性迴歸模型 (Multiple Linear Regression)」，使用公司各項支出與所在州別來預測 Profit。

📊 目標變數與輸入特徵

  • 目標變數 (y): Profit (公司獲利)
  • 連續型特徵 (X):
    • R&D Spend (研發支出)
    • Administration (行政支出)
    • Marketing Spend (行銷支出)
  • 類別型特徵 (X): State (所在地區: New York, Florida, California)

4. Data Preparation 資料準備 Slide 4 / 8

🏷️ One-Hot Encoding (獨熱編碼)

類別特徵 State 包含：New York, Florida, California。迴歸模型無法直接讀取字串，因此進行獨熱編碼轉換。

⚠️ 避免 Dummy Variable Trap

當我們把一個包含 k 個類別的特徵全部獨熱編碼時，各虛擬變數加總恆等於 1，這會與截距項形成完美共線性。為此我們丟棄 California 作為 baseline，僅保留 New York 與 Florida。若兩者皆為 0 即代表 California。

✂️ 訓練集與測試集切分

  • 使用 train_test_split 設定。
  • 測試集比例 (test_size) 為 20% (10 筆測試，40 筆訓練)。
  • 設定固定亂數種子 random_state = 42。
5. Modeling 模型建置與特徵數對比 Slide 5 / 8

比較五種不同的特徵組合，在測試集上的預測能力：

模型與特徵數 包含特徵 RMSE R-squared Adj. R-squared
1 Feature R&D Spend 7714.33 0.9265 0.9173
2 Features (最佳) R&D Spend, Marketing Spend 8206.33 0.9168 0.8931
3 Features R&D, Marketing, New York 8242.78 0.9161 0.8741
4 Features R&D, Marketing, New York, Florida 8254.69 0.9159 0.8485
5 Features R&D, Marketing, New York, Florida, Admin 9055.96 0.8987 0.7721

💡 重要洞察：隨著加入特徵數從 2 個增加到 5 個，測試集 RMSE 卻由 8206 增加到 9055，Adjusted R-squared 由 0.89 大幅縮減到 0.77。這說明特徵並非越多越好，過多特徵可能造成過擬合 (Overfitting)。

7. Evaluation & Saved Model 模型評估 Slide 7 / 8

🎯 最終模型參數 (2-Feature Model)

  • 截距 (Intercept): 50286.8118
  • R&D Spend 係數 (Coefficient): 0.8056
  • Marketing Spend 係數 (Coefficient): 0.0272
Profit = 50286.8118 + 0.8056 * (R&D Spend) + 0.0272 * (Marketing Spend)

🔍 模型係數意義

研發支出每增加 1 元，預估利潤增加 0.81 元；行銷支出每增加 1 元，利潤增加 0.03 元。資源應大幅度優先挹注於研發項目。

8. Deployment & Streamlit 部署應用 Slide 8 / 8

🔮 Streamlit 互動預測平台

使用 Streamlit 架設了預測預算分配的 Web 應用。使用者可直接利用拉桿調整「研發預算」與「行銷預算」，網頁將在毫秒級內更新並輸出預測 Profit。

📈 商業決策應用

  • 研發推動型：初創新創公司應降低管理行政摩擦，將主要盈餘循環投資於 R&D 以換取長期的高獲利增長率。
  • 行銷邊際效應：行銷可作為輔助工具，但不應作為核心的獲利推動引擎。
